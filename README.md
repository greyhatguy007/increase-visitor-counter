A simple code to increase the visitor count in github


## Dependencies:
  - python3 - selenium
  - broswer - chrome
  - chrome webdriver

## Installation :

* git clone https://github.com/greyhatguy007/increase-visitor-counter.git
* cd increase-visitor-counter/
* pip install -r requirements.txt
* python3 increase.py
